What is this?

Sd-calendar is a reusable date-picker component that can be easily integrated into any Angular 2 application. It was created with Angular 2 and Typescript and extends the PrimeNG calendar component by adding the following features:

  • Input masking to prevent basic user errors
  • Automatic formatting to ensure the underlying model is always valid
  • Optional date range restriction

What problem does this fix?

The client's inputs were broken in Safari and their entire application was unusable in IE 11. The problem was the use of input type="date" inputs, which have poor browser support.

Sd-calendar fixes these problems, while dramatically improving the user experience and adding error prevention.

How do I view the Source Code?

The source code for sd-calendar can be found under src > app > calendar, and consists of the following 4 files:

sd-calendar-input.component.ts, sd-calendar-input.template.html, sd-calendar-input.style.scss, and utils.service.ts.

All other files in the repository were generated by Angular CLI
